Introduction

Super POM for my Projects. The POM contains various settings which I frequently use.

Installation

Release builds are available from the Maven repository:

  http://dl.bintray.com/javanarior/maven

To use the POM set the parent like

    <parent>
        <groupId>de.javanarior</groupId>
        <artifactId>javanarior-oss-pom</artifactId>
        <version>40</version>
    <parent>
